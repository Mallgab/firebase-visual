About the Data

This website visualizes findings from the dataset “Purposes for Internet Usage by Individuals.”

The data record how people use the internet—for example for communication, education, entertainment, online shopping, or e-government services—broken down by gender, age group, region, and year.

The original figures come from official national statistics portal.

Values are given as percentages of individuals in each demographic group who reported using the internet for each purpose.

Dataset is used to create interactive charts that show:

  • differences between men and women,
  • patterns across age groups,
  • changes over time,
  • and regional comparisons.
